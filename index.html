To set up visual studo code, you first have download it from this link:https://code.visualstudio.com/Download. Then you have to create a folder on your computer/laptop, and open Visual Studio Code. Once you have opened Visual Studio Code, open the explorer section and click on open folder. Select the folder you created and create a file called index.html in Studio Visual Code.

The console screen can be opened by right clicking on a web page. Once you have right clicked on the web page, clcik on inspect and a section will appear on the right side of the page. Look for console, on the top part of that section and click on it to acces the console screen. The console screen is usually used to store variables from your code. To print things on the console screen use: console.log(). Example: console.log(first_name).
